Альтернатива «СтопХаму» // Перед вокзалом города Сочи построили новую автостоянку

2012-10-21

Это Вам не какой-нибудь СтопХам и даже не флэшмоб вовсе. Перед вокзалом появилась новая парковка для автомобилей. На тот момент, когда я делал эти фотографии, техник монтировал шлагбаум для въезда с улицы Островского. Пока же на парковку можно попасть только с Московской. На данный момент парковка бесплатная…

Техник монтирует шлагбаум для въезда на парковку автомобилей с улицы Островского. 

Судя по этим терминалам, вскоре оплата за использование парковки будет происходить по карточкам.
