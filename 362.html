Участникам олимпийской стройки предложат льготную ипотеку

2012-10-21

В Сочи готовится льготная ипотечная программа для постоянного персонала проекта «Сочи-2014». Как сообщила во вторник пресс-служба Кремля, об этом накануне на встрече с президентом РФ Дмитрием Медведевым рассказал вице-премьер Дмитрий Козак. По словам Козака, для персонала Олимпиады подготовлен целый ряд программ: «это и программа обеспечения безопасности, медицинского обеспечения, программа олимпийского наследия и программа по привлечению, размещению и подготовке персонала для проведения Олимпийских игр и для обслуживания тех объектов, которые вновь создаются». © фото из архива www.BlogSochi.ru Как сообщил вице-премьер, в пиковый период 2013–2014 годов в Сочи будут привлечены более 100 тыс. человек, половина из них – это волонтеры, то есть временные сотрудники, которые будут привлекаться в основном оргкомитетом для обеспечения Олимпийских игр. При этом, как рассказал Козак, в первую очередь будут привлечены жители Сочи, затем – жители Краснодарского края, а затем — жители других регионов страны. При этом речь идет, прежде всего, о студентах. По его словам, готовятся обучающие программы, которые предполагают обучение языку, навыкам обслуживания на мировом уровне. «По поручению Правительства России городом (Сочи) совместно со Сбербанком и банком ВТБ готовится льготная ипотечная программа для того персонала, который привлекается на постоянной основе для работы, для тех, кто может приобрести жилье с помощью ипотеки. Для тех, кто не может, мы понимаем, что такие люди будут, которые будут привлекаться городом, готовятся предложения для работодателей по схеме размещения этих людей в каком-то социальном жилье, которое работодатели должны будут реализовывать», — рассказал Дмитрий Козак. Он отметил, что для привлечения людей будут использоваться краевые органы занятости, которые будут мониторить рынок труда и в Сочи, и в Краснодарском крае, для того чтобы направлять сюда людей и во взаимодействии со всеми службами занятости других регионов привлекать лучший персонал для обслуживания олимпийских объектов и других объектов, которые здесь создаются. www.rosbalt.ru
