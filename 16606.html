Прием граждан заместителем начальника УВД по городу Сочи

2014-05-13

Сообщение пресс-службы УВД Сочи:

В период с 13 по 30 мая 2014 года в отделах полиции УВД по городу Сочи будет осуществляться прием граждан заместителем начальника УВД по городу Сочи – начальником полиции полковником полиции Огурцовым Сергеем Ивановичем.

13 мая 2014 года в 17.00 прием граждан состоится в отделе полиции (Лазаревский район) УВД по г. Сочи по адресу:

16 мая 2014 года в 17.00 прием граждан состоится в отделе полиции (Центральный район) УВД по г. Сочи по адресу: г. Сочи, ул. Горького, 60;

20 мая 2014 года в 17.00 прием граждан состоится в отделе полиции (Хостинский район) УВД по г. Сочи по адресу: г. Сочи, ул. Аллея Челтенхема, 8А;

23 мая 2014 года в 17.00 прием граждан состоится в отделе полиции (Адлерский район) УВД по г. Сочи по адресу: г. Сочи, ул. Кирпичная, 1А;

27 мая 2014 года в 17.00 прием граждан состоится в отделе полиции (п. Блиново) УВД по г. Сочи по адресу: г. Сочи, ул. Лесная, 20А;

30 мая 2014 года в 17.00 прием граждан состоится в отделе полиции (пгт. Красная поляна) УВД по г. Сочи по адресу: г. Сочи, ул. Озерная, 1.

Предварительная запись проводится с 15.00 до 17.00 по телефону: 269-71-01.
