Татьяна Навка провела мастер класс для воспитанников школы зимних видов спорта Сочи

2012-05-15

Олимпийская чемпионка по фигурному катанию, посол "Сочи-2014" Татьяна Навка провела мастер класс для воспитанников школы зимних видов спорта Сочи на льду строящегося дворца спорта "Айсберг" в Олимпийском парке Сочи.
