Информация о плановых отключениях электроэнергии

2016-10-27

В связи с планово-предупредительным ремонтом  в рамках текущей эксплуатации электрооборудования возможно ограничение подачи электроэнергии на перечисленных ниже улицах. Данные мероприятия необходимы для предупреждения технологических нарушений и призваны обеспечить надежное электроснабжение

28 октября  с 04.00 до 17.00  (в течение нескольких часов из указанного временного промежутка): с.Галицино, с.Монастырь, лососевый завод, центр разведения и реабилитации переднеазиатского леопарда.

28 октября  с 08.00 до 17.00  (в течение нескольких часов из указанного временного промежутка): в селе Верхне-Николаевское: ул.Междуреченская, ул.Николаевская, ул.Мостовая, ул.Псебайская,  ул.Тульская

28 октября  с 09.00 до 18.00 (в течение нескольких часов из указанного временного промежутка): ул.Главная, ул.Придорожная, переулок Придорожный

28 октября  с 09.00 до 18.00 (в течение нескольких часов из указанного временного промежутка): ул.Торговая

28 октября  с  08.00 до 17.00 (в течение нескольких часов из указанного временного промежутка): ул.Дмитриевой №№7-12, котельная №38,

28 октября  с  08.00 до 17.00 (в течение нескольких часов из указанного временного промежутка): ул.Первомайская №№ 2, 3а, 4, 4а, 6; Курортный проспект № 29, кафе «Горка», художественный салон, АО «Полюс»

28 октября  с  08.00 до 17.00  (в течение нескольких часов из указанного временного промежутка): ул.Пластунская (частично), ул.Вишневая №5, ТСЖ «Вишневый», ООО «Отделстрой», ОАО «НК Роснефть»,

28 октября   с  13.00 до 16.00 (в течение нескольких часов из указанного временного промежутка): ООО «Транс Строй» на ул.Пластунская
