Колонка главреда // Борьба с троллингом на проекте

2011-12-26

Для создания на сайте более полноценной дискуссии вводится ряд мер для борьбы с троллями, то есть с теми пользователями сайта, которые пользуются вымышленным именем и провоцируют конфликты между пользователями.

  • Во-первых, с этого момента новым пользователям сайта будет отказано в регистрации на сайте, если они не разместят свою фотографию (исключительно портрет) и кратко не опишут свой род деятельности.
  • Во-вторых, редакционная коллегия вправе принять решение о заблокировании пользователя, если он откажется как-либо подтвердить свою личность, как правило, будет достаточно продемонстрировать свой аккаунт в популярных соцсетях.
  • В-третьих, пользователям не рекомендуется с этого момента использовать в дискуссии чрезмерное кол-во массивных текстов. Такие действия будут расцениваться как флуд, злоупотребление которым может повлечь блокирование аккаунта.
  • В-четвертых, флудом будет считаться переход на личности и какие-либо личные разборки на страницах проекта.
  • В-пятых, на проекте запрещено фамильярничать и обращаться на «ты» к пользователям, которые такое обращение не приемлют.
Нарушение правил дискуссии на проекте может также повлечь ограничение прав аккаунта и запрет на комментарии на проекте.
