Окончена установка кровли Ледовой арены для кёрлинга

2011-10-05

В Имеретинской низменности завершились работы по устройству кровли Ледовой арены для кёрлинга.

На несущие конструкции покрытия здания уложены 3 слоя. Сначала металлический каркас крыши закрыли профилирующим настилом, со стороны помещения его окрасили в серо-белый цвет. Эту стальную основу утеплили теплоизоляционными плитами из минеральной ваты. Для защиты от дождевых и талых вод был уложен слой гидроизоляции.

Крышу площадью более 1000 квадратных метров оборудовали приточно-вытяжными установками. Они предназначены для удаления отработанного воздуха и забора свежего кислорода, его фильтрации, охлаждения, подогрева и раздачи с помощью воздуховодов по помещениям.

Очередным этапом работ станет монтаж декоративных фасадов здания. Для обшивки будут применяться сэндвич-панели с утеплителем и декоративные металлические панели.

Ледовая арена для керлинга представляет собой сборно-разборную конструкцию и возведена на железобетонной фундаментной плите. На нижнем уровне спортивного сооружения разместится ледовое поле с игровыми площадками, раздевалки, служебные зоны для атлетов, помещения для судей, врачей, представителей СМИ, технические и складские помещения.

По периметру помещения будут построены трибуны для трех тысяч зрителей.

Арену сдадут в эксплуатацию в 2012 году. Ответственный исполнитель - ООО «Инвестиционно-строительная компания «Славобласть».
