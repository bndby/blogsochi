ДСД "Черноморье": Начата реконструкция улицы Клубничной

2012-11-20

Начата реконструкция проезжей части улицы Клубничной в районе строительства 3 очереди будущей основной транспортной артерии города-курорта «Дублер Курортного проспекта. Работы по укреплению дорожного полотна важны в связи с необходимостью проезда по ней технологического транспорта большой грузоподъемности. По плану процесс реконструкции должен быть завершен в течение месяца, но дорожники прилагают максимум усилий, чтобы выполнение работ завершилось досрочно.

«Состояние дорожного покрытия находилось в неудовлетворительном состоянии, не обеспечивало возможности движения грузового транспорта», – сообщает пресс-служба Дирекции по строительству дорог «Черноморье». - По окончанию строительных мероприятий проезд по обновленной дороге для легкового автотранспорта жителей микрорайона станет более комфортным».
