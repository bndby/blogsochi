Колонка главреда // «Вставить из Word» и смайлики ))))

2010-10-03

Ранее на сайте существовала проблема: при вставке из ворда в тексте появлялись огромные пробелы, а шрифт букв был слишком большим. Теперь эту проблему легко обойти. Для этого нужно при вставке текста из ворда использовать кнопку «Вставить из Word» в панели инструментов. Также в панели появились еще и другие кнопки из стандартного набора ЖЖ, которые могут облегчить работу с текстом. Надеюсь новшество всем понравится. и теперь на сайте есть смайлики ...
