Следственный комитет возбудил уголовное дело по факту ДТП с участием сотрудников УВД по городу Сочи

2015-11-23

Следственный комитет Российской Федерации от 23.11.2015:

Следственным отделом по Центральному району города Сочи СКР по краю по результатам проведенной проверки по факту произошедшего дорожно-транспортного происшествия с участием автомобилей УАЗ Патриот и БМВ, в результате которого погиб один из водителей, возбуждено уголовное дело по признакам преступления, предусмотренного ч.3 ст.264 УК РФ (нарушение правил дорожного движения, повлекшее по неосторожности смерть человека)...

По версии следствия, 16 ноября 2015 года во второй половине дня служебный автомобиль отдела полиции УАЗ Патриот под управлением сотрудника ППСП отдела полиции (Центральный район) УВД по городу Сочи двигался со стороны улицы Чайковского по улице Новая Заря с включенным проблесковыми маячками по полосе встречного движения. В это же время по дороге ехал автомобиль БМВ, которым управлял не находившийся при исполнении служебных обязанностей сотрудник этого же отдела. В результате произошедшего столкновения указанных автомобилей водитель БМВ получил травмы, от которых скончался.

В настоящее время проводятся следственные действия, направленные на установление всех обстоятельств совершенного преступления. Расследование уголовного дела продолжается.
