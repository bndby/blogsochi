Городские цветы

2011-11-25

Сегодня в администрации города руководитель управления по охране окружающей среды встретился с представителями дирекции заказчика всех районов и управления архитектуры. Речь шла об изменении некоторых пунктов в правилах по содержанию зеленых насаждений.

Чтобы олимпийская столица расцвела, по поручению главы города Анатолия Пахомова, создана специальная комиссия по озеленению курорта. Ее сотрудники определят места для будущих скверов и согласуют проекты по созданию новых парков и реконструкции уже существующих.

Работа комиссии уже дает свои результаты. Так, например, около набережной в микрорайоне Мамайка появились несколько сотен саженцев. В планах - благоустроить сквер на противоположном берегу реки. Такому «зеленому» подарку от градоначальника, жители одного из самых чистых районов города рады.

Активисты сами взялись за лопаты и грабли.

Следующая точка — Лазаревский район. Возле домов для переселенцев по улице Калараша сейчас нет мест для отдыха. Но уже совсем скоро, как только будет согласован проект, начнутся работы по устройству сквера. Озеленение коснется и остальных районов Большого Сочи.
