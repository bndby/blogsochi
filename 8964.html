Открытое обращение членов комиссии по общественным слушаниям по ОВОС Кудепстинской ТЭС к Президенту Российской Федерации, Главе Администрации Краснодарского края, главе администрации г.Сочи, жителям Сочи.

Почему разработанный проект ОВОС (оценка воздействия на окружающую среду) по Кудепстинской ТЭС не прошёл общественные слушания?...

Исходя из анализа выступлений независимых от администрации специалистов и экспертов по экологии и жителей Кудепсты на общественных слушаниях по ОВОС Кудепстинской ТЭС считаем, что представленный к слушаниям отчет по ОВОС не соответствует действительности как по количеству вредных выбросов, так и по использованию исходных данных для ОВОС и по ряду других параметров.

На Кудепстинской ТЭС планируется установить 20 газо-поршневых двигателей 18V50SG производства финской корпорации Wartsila, общей мощность 367 мгв.

Количество выбросов вредных веществ указанных в представленном отчете по ОВОС Кудепстинской ТЭС - неверны, т.к расчеты приведены для котельных установок, а не для газо-поршневых двигателей, а также отсутствует составляющая по выгоранию масла - 1376 т / год

Исходные данные для расчета рассеивания вредных веществ в атмосферу в ОВОСе Кудепстинской ТЭС взяты не для территории выбранной площадки, а для равнинной местности.

Климатическая характеристика, взятая в расчете по ОВОС, является общей для Адлерского района.

Коэффициент рельефа местности, взятый в расчетах , не соответствует действительности для выбранной площадки под ТЭС.

Расчеты рассеивания вредных веществ от Кудепстинской ТЭС, проведенные институтом ИМЭПОРЗ, выполнены не для Кудепсты, а для другой местности.

Считаем, что данный отчет по ОВОС Кудепстинской ТЭС не может быть направлен на редактирование, т.к. он не соответствует действительности.

Необходимо:

1. Разработать новый проект по ОВОС для Кудепстинской ТЭС, предварительно произведя по действующим положениям полный мониторинг местности по гидрометеоусловиям в течении одного-двух лет.

2. Рассчитать объективные исходные данные по выбросам вредных веществ для газо-поршневых двигателей

3. Сделать расчет труб и фильтров с учетом рельефа местности и количества выбрасываемых веществ по методике ОНД-86 в соответствии с действующими положениями.

4. Предоставить жителям Кудепсты и Сочи для изучения вновь составленный отчет по ОВОС для Кудепстинской ТЭС.

5. В сроки, установленные законодательством провести общественные слушания и уже, затем принять решение о строительстве.

Члены комиссии : И.Васильев, Е.Егоров, Л.Хворов, А.Махновский, С.Беседин, А.Мартыненко
