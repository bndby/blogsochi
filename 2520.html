«Черно-белый Сочи» // Спасем улицу Чайковского!!!

2011-02-10

Предлагаю всем фотографам Сочи поучаствовать во флэшмобе «Черно-белый  Сочи», цель которого привлечь внимание к вырубке платанов по улице Чайковского. Для участие во флэшмобе достаточно публиковать в Интернете фотографии с улицы Чайковского исключительно черно-белые. Данный пост прошу считать началом флэшмоба. Акция будет завершена только тогда, когда компания «Мостотрест» полностью компенсирует нанесенный природе урон компенсационными посадками, которые удовлетворят большинство общественности города.
